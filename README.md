# TranslateThis

Frontend for TranslateThis made using Vite, React, Typescript, Zustand and DaisyUI (tailwindCSS plugin)
Backend: https://github.com/KishanBapodra/translate-this-backend/  

### Install the required dependancies

```
npm install
```

### Run the Server

```
npm run dev
```
